## Usage

First, git clone the whole depository using command : 

`git clone https://github.com/Minh42/42_project-fillit [name]`

Run makefile to compile using command:  

`make`

Run executable using command:  

`./fillit [file]`

Example:
```
~ ./fillit samples/test_8

.AABB..
AACCB..
.CC.BDD
.EE.DD.
EEFFGG.
.FFGGHH
....HH.
```
